Tracking Examples

Below are JS examples you can deploy inside your templates or add to a sitewide script. They push structured events to the dataLayer for Tag Manager to pick up.

// Page view (send on page load)
window.dataLayer = window.dataLayer || [];
window.dataLayer.push({
  event: 'page_view',
  page_path: location.pathname,
  page_title: document.title,
  site_section: 'marketing'
});

// CTA click example
function trackCTAClick(label){
  window.dataLayer.push({
    event: 'cta_click',
    cta_label: label,
    page_path: location.pathname
  });
}

// Form submit example (call after successful submit)
function trackFormSubmit(formName, success){
  window.dataLayer.push({
    event: 'form_submit',
    form_name: formName,
    form_success: !!success
  });
}

// E-commerce product view example
function trackProductView(product){
  window.dataLayer.push({
    event: 'view_item',
    ecommerce: {items: [product]}
  });
}

Implementation & QA Checklist

Copy these into your deployment checklist
  • Install GTM container snippet on all pages (head + body).
  • Install GA4 via GTM (use Measurement ID in GA4 tag or use gtag if preferred).
  • Verify dataLayer pushes appear in preview mode for each event type.
  • Test key conversion events: contact form, phone click, brochure download, product view, add-to-cart.
  • Set up appropriate GA4 events & conversions via GTM or directly in GA4.
  • Test cross-domain and referral exclusions if using external payment or partner flows.
  • Document event naming, parameters and owner for each event.
  • Confirm data retention and user-ID strategy with legal/privacy team.
